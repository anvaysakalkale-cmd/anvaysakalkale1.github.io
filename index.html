CNC Machined Part Inspection System

WIZER Machine Vision Baumer Camera Python Excel Dashboard Industry 4.0
  • Configured WIZER — a no-code industrial machine vision platform — on a Baumer industrial camera to perform automated real-time surface defect detection on CNC machined steel parts (scratches, cracks, inclusions, pitting, crazing).
  • Trained the defect detection module using 50–100 reference images of defect-free parts, building a statistical surface model; tuned sensitivity thresholds to balance detection accuracy against false-positive rate.
  • Integrated a Python-based data logging module that reads WIZER output and records defect type, location, confidence score, and Accept/Reject verdict into a structured Excel workbook with a live inspection dashboard.
  • Delivered a fully deployable, no-code inspection station suitable for CNC machining workshops under Industry 4.0 quality requirements — requiring no deep learning expertise to operate or maintain.

Mitsubishi PLC-Based Automated Bottling Plant

Mitsubishi PLC HMI Ladder Logic GX Works Sensors & Actuators
  • Designed and implemented a fully automated bottling plant using a Mitsubishi PLC integrated with HMI, proximity/photoelectric sensors, solenoid valves, timers, counters, and a conveyor system.
  • Programmed ladder logic for automated bottle detection, conveyor control, timed solenoid filling, pneumatic feed-separator control, and post-fill bottle counting using diffuse sensors.
  • Implemented PLC arithmetic function blocks to calculate total bottles filled, remaining batch quantity, and average bottles per cycle — all displayed in real time on the HMI.
  • Achieved uniform filling, accurate counting, and safe operation with minimal human intervention — improving production accuracy, cycle time, and operator safety.
